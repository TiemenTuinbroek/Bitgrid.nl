Wat is een Discord BOT

Discord-bots zijn AI's die een aantal nuttige geautomatiseerde taken en Discord-opdrachten op uw server kunnen uitvoeren, zoals het verwelkomen van nieuwe leden, het modereren van inhoud en het verbieden van regelovertreders. U kunt Discord-botopdrachten gebruiken om muziek, memes, games en andere inhoud aan uw server toe te voegen.

Wat kan de Discord BOT?

-De BOT luistert naar tekst commands
